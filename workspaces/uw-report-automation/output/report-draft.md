| 집계 기간 | 월별 추이 최소 12개월 권장 / 실데이터 제공 시 확정 |

**이상 감지 임계값 확정 현황**

### 2.5 M-04-EX 사고청구 분석 (M-04 확장 모듈)

> M-04-EX는 M-04 보험취급자 위험 관리 모델의 확장 모듈로, 사고청구 원천 데이터를 집계(EX-1)하고 이상 청구를 탐지(EX-2)한다. 설계사코드(agent_code) 기준으로 M-04와 조인하여 연계 분석한다.

| 항목 | 내용 |
|---|---|
| 모델 유형 | 집계 기반 현황 분석(EX-1) + 규칙 기반 이상 탐지(EX-2) |
| 집계 단위 | 설계사코드(agent_code) / 대리점코드(agency_code) / 채널 / 상품군 / 담보군 / 청구월 |
| M-04 연계 키 | agent_code (설계사코드) |
| 원천 컬럼 수 | 67개 (비식별 처리: PII 마스킹 및 자유텍스트 → 키워드 플래그 대체) |
| 설계 완료 여부 | 설계 완료 (탐지 임계값은 실데이터 기반 확정 예정) |

**EX-1 핵심 집계 지표**

| 지표명 | 계산 방식 | 분석 목적 |
|---|---|---|
| 사고건수 | COUNT(사고영수번호) | 설계사 단위 청구 빈도 파악 |
| 총보험금 | SUM(보험금) | 설계사 단위 손해 볼륨 파악 |
| 건당보험금 | 총보험금 / 사고건수 | 건당 심도 이상 감지 |
| 피보험자 반복청구율 | 다발청구 피보험자수 / 총 피보험자수 × 100 | 피보험자 다발 청구 패턴 감지 |
| 면책·환수 비율 | 면책건수 / 총 청구건수 × 100 | 부정 청구 가능성 지표 |
| 담보군별 보험금 비중 | 담보군 보험금 / 총보험금 × 100 | 특정 담보 집중 이상 감지 |
| 청구빈도 | 사고건수 / 유효계약수 | 계약 대비 청구 과도 여부 |

**EX-2 탐지 기준 현황**

> 근접사고 기준(DT-1)은 상품군별 면책·감액 기간을 반영하여 DT-1a~1e로 세분화. 치아보험 감액 비율은 약관 확인 후 확정 예정.

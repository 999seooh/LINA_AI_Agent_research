## M-01 손해율 분석 대시보드

### 1. 대시보드 개요

| 항목 | 내용 |
|---|---|
| 대시보드명 | M-01 손해율 분석 현황 |
| 목적 | 기간별·상품별 손해율 추이 모니터링 및 이상 구간 조기 경보 |
| 갱신 주기 | 월 1회 (계약연월 기준 월말 마감 후 / 실데이터 제공 시 확정) |
| 주요 사용자 | UW 담당자, UW 팀장 |
| 데이터 출처 | 원천: earned_premium, incurred_loss, claim_count (마스킹 데이터) |

### 2. 지표 정의

| 지표명 | 계산 방식 | 집계 단위 | 필터 조건 | 시각화 유형 |
|---|---|---|---|---|
| 월별 손해율 | incurred_loss / earned_premium × 100 | product_code × contract_ym | earned_premium > 0 | 꺾은선 그래프 |
| 손해율 전월 대비 변화율 | (당월 LR - 전월 LR) / 전월 LR × 100 | product_code × contract_ym | earned_premium > 0 | 막대 그래프 (증감 색상 구분) |
| 3개월 이동평균 손해율 | 최근 3개월 손해율 단순 평균 | product_code 단위 | earned_premium > 0 | 꺾은선 그래프 (추세선) |
| 12개월 누적 손해율 | 12개월 누적 incurred_loss / 누적 earned_premium | product_group 단위 | earned_premium > 0 | 수평 막대 그래프 |
| 청구 빈도 | claim_count / 계약건수 | product_code × contract_ym | 유효 계약 기준 | 꺾은선 그래프 |
| 이상 감지 플래그 건수 | 임계값 초과 product_code 건수 | 월별 | 전체 | 숫자 카드 (이상 건수) |
| 채널별 손해율 비교 | 채널별 incurred_loss / earned_premium × 100 | channel_code × contract_ym | earned_premium > 0 | 그룹 막대 그래프 |

### 3. 화면 구성안

| 영역 | 섹션명 | 주요 내용 | 위치 |
|---|---|---|---|
| 상단 요약 | 핵심 KPI 카드 | 당월 전체 손해율 / 전월 대비 변화율 / 이상 감지 건수 / 12개월 누적 손해율 | 최상단 |
| 중단 (탭 1) | 손해율 추이 | 월별 손해율 꺾은선 (상품군 선택 가능) + 3개월 이동평균 중첩 | 중간 좌 |
| 중단 (탭 1) | 이상 감지 현황 | 임계값 초과 상품코드 목록 + 플래그 유형 표 | 중간 우 |
| 중단 (탭 2) | 상품별 분석 | product_code × 손해율 히트맵 (기간별) | 중간 전체 |
| 하단 (탭 3) | 채널별 비교 | 채널별 손해율 그룹 막대 그래프 | 하단 좌 |
| 하단 (탭 3) | 청구 빈도 추이 | 청구 건수 증감 꺾은선 | 하단 우 |
| 하단 (탭 4) | 상세 집계 표 | product_code × contract_ym 단위 전체 집계 데이터 테이블 (정렬: 손해율 DESC) | 하단 전체 |

### 4. 차트 유형 및 데이터 연결

| 차트명 | 차트 유형 | X축 | Y축 / 값 | 색상 구분 |
|---|---|---|---|---|
| 손해율 추이 | 꺾은선 (다중 계열) | contract_ym (월) | loss_ratio (%) | product_group별 색상 |
| 이동평균 추이 | 꺾은선 (단일) | contract_ym (월) | 3개월 이동평균 손해율 | 기준선 (목표치) 별도 표시 |
| 상품별 히트맵 | 히트맵 | contract_ym (월) | product_code | 셀 색상: 손해율 수준 (저→고: 녹→적) |
| 채널 비교 | 그룹 막대 | contract_ym (월) | loss_ratio (%) | channel_code별 색상 |
| 이상 감지 플래그 | 산점도 또는 표 | contract_ym | product_code | 플래그 유형별 마커 |

### 5. 필터·드릴다운 기준

| 상품군 (product_group) | 다중 선택 드롭다운 | 손해율 추이, 히트맵 |

## M-02 유지율 분석 대시보드

> **[v1.1 변경 사항]** 구 설계(lapse_yn·lapse_reason_code 기반 생존 분석)를 폐기하고,
> 실제 SQL 구현(MMP 기반 코호트 유지율 집계)에 맞춰 전면 재설계. SQL 버그 수정 Fix-1~7 반영.
> 채널 구조: `channel = 'DM'`(상위 필터) / `channel_1`(GA·CM·OB 세분화) 기준으로 통일.

### 1. 대시보드 개요

| 항목 | 내용 |
|---|---|
| 대시보드명 | M-02 유지율 분석 현황 |
| 목적 | 청약월 코호트별 MMP 기준 유지율 집계, 주요 회차(4·7·13·25회차) 이탈 패턴 추적, 채널·상품별 유지 성과 비교 및 손해 연계 리스크 관리 |
| 갱신 주기 | 월 1회 (baseYM 기준 월말 마감 후 / 실데이터 제공 시 확정) |
| 주요 사용자 | UW 담당자, UW 팀장 |
| 채널 범위 | DM 채널 전체 (GA·CM·OB 포괄) — `channel = 'DM'` 필터 기준 |
| 원천 테이블 | YM_PersistencyRatio, V_GDMS_Contract, DW_productcode_table, V_GDMS_MF_ClaimDetail, V_GDMS_MF_PremiumDetail_LT, DW_plancode_table |
| 주요 입력 컬럼 | baseYM, appMonth, tenure, salesValue(모집MMP), value(유지MMP), channel_1, 상품코드·상품군, 플랜구분, 담보군·담보구분, 연령대, riskP, GWP, loss_Amt |

### 2. 지표 정의

**MMP 기반 유지율 지표**

| 지표명 | 계산 방식 | 집계 단위 | 필터 조건 | 시각화 유형 |
|---|---|---|---|---|
| MMP 기준 유지율 | 유지MMP / 모집MMP × 100 | 청약월(코호트) × 기준년월 × 채널 × 상품코드 | 모집MMP > 0, tenure ≠ 0 | 코호트 매트릭스 (히트맵) |
| 유지율 전월 대비 변화 | 당월 유지율 - 전월 유지율 | 기준년월 단위 | 전체 | 꺾은선 그래프 (증감 색상) |
| 이탈 MMP | 모집MMP - 유지MMP | 청약월 × 기준년월 | 전체 | 막대 그래프 |
| 코호트 누적 유지율 | 청약월 기준 기준년월별 유지MMP / 모집MMP × 100 | 청약월 코호트 단위 | 전체 | 코호트 꺾은선 |

**주요 회차별 유지율 지표**

| 지표명 | 계산 방식 | 집계 단위 | 필터 조건 | 시각화 유형 |
|---|---|---|---|---|
| 4회차 유지율 | 유지MMP / 모집MMP × 100 (tenure=4) | 청약월 코호트 | 회차구분 = 주요회차 | 꺾은선 + 임계값 기준선 |
| 7회차 유지율 | 유지MMP / 모집MMP × 100 (tenure=7) | 청약월 코호트 | 회차구분 = 주요회차 | 꺾은선 + 임계값 기준선 |
| 13회차 유지율 | 유지MMP / 모집MMP × 100 (tenure=13) | 청약월 코호트 | 회차구분 = 주요회차 | 꺾은선 + 임계값 기준선 |
| 25회차 유지율 | 유지MMP / 모집MMP × 100 (tenure=25) | 청약월 코호트 | 회차구분 = 주요회차 | 꺾은선 + 임계값 기준선 |

**채널·상품별 성과 지표**

| 지표명 | 계산 방식 | 집계 단위 | 시각화 유형 |
|---|---|---|---|
| 채널별 MMP 유지율 | 유지MMP / 모집MMP × 100 | 채널(channel_1) × 청약월 × 기준년월 | 그룹 꺾은선 |
| 상품군별 유지율 | 유지MMP / 모집MMP × 100 | 상품군 × 청약월 코호트 | 그룹 막대 |
| 대리점별 유지율 | 유지MMP / 모집MMP × 100 | agency명 → center명 드릴다운 | 수평 막대 (상위 N) |

**손해 연계 지표**

| 지표명 | 계산 방식 | 집계 단위 | 시각화 유형 |
|---|---|---|---|
| 손해율 (riskP 기준) | loss_Amt / riskP × 100 | 청약월 코호트 × 채널 | 꺾은선 (유지율 동시 표시) |
| 총지급보험금 대비 모집MMP | 총지급보험금 / 모집MMP × 100 | 청약월 코호트 | 꺾은선 |
| 유지율 하위 코호트 손해율 | 유지율 하위 N% 코호트의 loss_Amt / riskP | 코호트 단위 | 산점도 (유지율 × 손해율) |

### 3. 화면 구성안

| 영역 | 섹션명 | 주요 내용 | 위치 |
|---|---|---|---|
| 상단 요약 | 핵심 KPI 카드 | 전체 MMP 유지율 / 전월 대비 변화 / 이탈 MMP 규모 / 가장 낮은 채널 유지율 | 최상단 |

| 중단 (탭 2) | 주요 회차 이탈 분석 | 4·7·13·25회차 유지율 코호트별 꺾은선 (임계값 기준선 중첩) | 중간 |
| 하단 (탭 3) | 채널·상품 비교 | 채널별 MMP 유지율 그룹 꺾은선 + 상품군별 유지율 비교 막대 | 하단 |
| 하단 (탭 4) | 손해 연계 분석 | 유지율 × 손해율 산점도 + 채널별 유지율-손해율 동시 추이 | 하단 |
| 하단 (탭 5) | 조직별 드릴다운 | agency명 → center명 단계 유지율 수평 막대 | 하단 |
| 하단 (탭 6) | 상세 집계 표 | 청약월 × 기준년월 × 채널 × 상품코드 단위 전체 지표 테이블 (정렬: 유지율 ASC) | 하단 전체 |

### 5. 필터·드릴다운 기준

| 상품군 | 다중 선택 드롭다운 | 상품군 | 채널·상품 비교 탭 |

#### 9.1 대시보드 개요

| 항목 | 내용 |

| 원천 테이블 | 사고청구 집계 테이블 (비식별 처리 완료, 개인식별정보 미포함) |
| 집계 단위 | 설계사코드(agent_code) / 대리점코드(agency_code) / 채널(channel) / 상품군(product_group) / 담보군(coverage_group) / 청구월(claim_month) |

| 담보군별 보험금 비중 | 담보군 보험금 / 총보험금 × 100 | 설계사코드 × 담보군 | |
| 피보험자별 반복청구율 | 다발청구 피보험자수 / 총 피보험자수 × 100 | 설계사코드 | 2건 이상 기준 |
| 면책·환수 비율 | 면책건수 / 총 청구건수 × 100 | 설계사코드 × 청구월 | |

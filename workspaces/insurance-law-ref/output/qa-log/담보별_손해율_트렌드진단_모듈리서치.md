## 1. 기법 비교표

| 기법 | 분류 | 원리 | 손해율 적용 방법 | 장점 | 단점 |
|---|---|---|---|---|---|
| **관리도 (SPC / X-bar, I-MR)** | 통계적 공정관리 | 평균·표준편차 기반 UCL/LCL 설정, 이탈점 탐지 | 월별 MLR을 타점, 3σ 이탈 시 경보 | 직관적, 현업 설명 용이 | 정규분포 가정, 소량 데이터 불안정 |
| **이동평균 (MA / 3·6·12개월)** | 시계열 평활 | 과거 N개월 단순 평균으로 추세 파악 | 12개월 MA와 실적 MLR 괴리 모니터링 | 구현 단순, 계절 완화 효과 | 반응 지연, 가중치 동일 |
| **EWMA (지수가중이동평균)** | 시계열 평활 | 최근값에 높은 가중치(λ) 부여 | λ=0.2~0.3 설정 후 EWMA 추세선 대비 이탈 감지 | MA보다 빠른 반응, 튜닝 가능 | λ 설정 주관적, 해석 복잡 |
| **STL 계절분해** | 시계열 분해 | Seasonal-Trend decomposition using Loess, 추세·계절·잔차 분리 | 담보별 계절 패턴 제거 후 순수 추세 추출 | 계절성 명확한 담보(실손·상해)에 효과적 | 데이터 24개월 이상 필요, 구현 복잡 |
| **Changepoint Detection (PELT·BOCPD)** | 구조 변화 탐지 | 시계열 내 평균·분산 급변 구간 탐지 | 손해율 수준 전환점(상품 개정·의료 환경 변화) 식별 | 자동화 적합, 구조 변화 명확히 포착 | 모수 민감, 소량 데이터 false alarm 多 |
| **ELR 비교** | 계리적 | Expected Loss Ratio = 요율 산출 시 가정 손해율 | 실적 MLR ÷ ELR = 진전배수로 요율 적정성 진단 | 요율 설계와 직접 연계 | ELR 갱신 주기에 따른 시차 발생 |
| **Development Triangle (클레임 진전)** | 계리적 | 사고연도×경과기간 삼각형, LDF 추정 | 미보고 손해(IBNR) 반영한 궁극 손해율 산출 | IBNR 포함 조기 신호 | 담보별 충분한 연도 누적 필요 |
| **Bornhuetter-Ferguson (BF법)** | 계리적 | 진전법(CL)과 ELR을 가중 결합 | 초기 연도(데이터 희소) 담보에 ELR 비중 높게 적용 | 희소 데이터 안정적, 신담보 적합 | ELR 품질 의존, 가중치 설계 필요 |

## 2. 주요 지표 정의표

| 지표 | 정의 | 산식 (개요) | 모니터링 활용 방식 |
|---|---|---|---|
| **MLR (실적손해율)** | 실제 발생 손해액 / 경과보험료 | 발생손해액 ÷ 경과보험료 × 100 | 담보별 월별 추이 타점, 임계치 대비 이탈 감지 |
| **ELR (예정손해율)** | 요율 산출 시 가정한 목표 손해율 | 요율 산출 모형상 가정치 | MLR/ELR 비율로 요율 적정성 상시 진단 |
| **LDF / CDF** | 손해액 진전계수 / 누적 진전계수 | 인접 경과기간 손해 비율 (누적) | Development Triangle에서 궁극 손해 추정에 사용 |
| **IBNR** | 발생미보고손해, 아직 청구되지 않은 손해 | 궁극 손해 추정액 - 현재 보고 손해 | 조기 손해율이 과소 계상될 때 IBNR 조정 후 비교 |
| **빈도 (Frequency)** | 계약 건당 발생 건수 | 사고 건수 ÷ 노출 계약 수 | 심도와 분리하여 사고 발생률 트렌드 파악 |
| **심도 (Severity)** | 사고 1건당 평균 손해액 | 총 손해액 ÷ 사고 건수 | 의료비 단가 상승 등 심도 주도 손해율 상승 판별 |
| **MLR/ELR 비율** | 실적 대비 예정 비율 | MLR ÷ ELR | 1.0 초과 시 요율 부족 신호, 경보 기준점으로 활용 |

> **빈도·심도 분리 분석 의의:** 손해율 상승 원인이 사고 건수 증가(빈도 주도)인지 건당 지급액 증가(심도 주도)인지를 구분하여 UW 대응책(가입 제한 강화 vs. 보험료 조정)을 다르게 설계할 수 있음.

## 3. 경보 기준 설계 가이드

### 3-1. 임계치 설정 방식

| 방식 | 내용 | 적합 담보 유형 |
|---|---|---|
| **3σ 규칙** | 역사적 평균 ± 3 표준편차 이탈 시 경보 | 데이터 충분한 주요 담보 (실손, 상해사망 등) |
| **백분위수 기반** | 역사적 분포 상위 90~95 퍼센타일 초과 시 경보 | 비대칭 분포 담보, 대형 담보 |
| **연속 N회 이탈** | 임계치 내이더라도 N개월 연속 상승 시 경보 | 완만하게 악화되는 담보 트렌드 탐지 |
| **MLR/ELR 비율 기준** | 비율 1.05 이상 → 주의 / 1.10 이상 → 경보 | 요율 연계 모니터링 |
| **YoY 변화율** | 전년 동월 대비 상승률 임계치 설정 | 계절성 강한 담보 |

### 3-2. 담보 규모별 차등 기준

| 구분 | 기준 | 임계치 수준 | 경보 방식 |
|---|---|---|---|
| **대형 담보** | 월 경과보험료 기준 상위 / 데이터 충분 | 3σ 또는 95 퍼센타일 | 자동 Alert + 정기 리뷰 |
| **중형 담보** | 중간 수준 노출 | MLR/ELR 1.08 + 연속 3개월 상승 | 담당자 수동 확인 트리거 |
| **소형/신규 담보** | 계약 건수·경과보험료 소량 | BF법 기반 추정 MLR 사용, ELR ±15% | 분기 단위 리뷰, 통계적 경보 비중 낮춤 |

> **핵심 원칙:** 소형 담보에 통계적 경보 기준을 그대로 적용하면 false alarm 과다 발생. 소형 담보는 정성 판단(UW 담당자 리뷰)과 계리 추정(BF법)을 병행하는 이중 구조 권장.

## 5. 실무 설계 고려사항

**소량 담보 데이터 희소성 처리**
- 계약 건수 30건 미만 또는 경과보험료 소량 담보는 통계적 임계치 대신 BF법 기반 추정 손해율 사용
- 유사 담보군(risk pool) 클러스터링으로 데이터 보강 후 경보 기준 공유 적용 검토
- 신규 담보는 최소 12~24개월 누적 후 독립 통계 기준 전환 원칙 명문화

**계절성·대형사고 이벤트 예외 처리**
- STL 또는 YoY 비교로 계절 효과 제거 후 트렌드 판단
- 태풍·집중호우·팬데믹 등 대형 이벤트 발생 월은 "이벤트 플래그" 부여, 트렌드 기준에서 제외 또는 별도 분석
- 이벤트 조정 후 기저 손해율(Underlying Loss Ratio) 별도 관리
- 이벤트 플래그 기준은 업계 공시 또는 협회 통계 기반으로 객관적 기준 수립

**UW 담당자 리뷰 프로세스 연계**
- 경보 발생 → 자동 알림(대시보드/이메일) → UW 담당자 정성 검토 → 조치 여부 기록 → 피드백 루프 구성
- 경보 등급 구분(주의/경보/긴급)에 따라 리뷰 기한과 에스컬레이션 기준 설계
- UW 담당자 판단 결과(오경보 판정 포함)를 로그로 축적하여 임계치 재조정에 활용
- 분기별 담당자-계리팀 공동 검토 세션으로 모델 파라미터 재검토

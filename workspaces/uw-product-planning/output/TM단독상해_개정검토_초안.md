## 2. 71~75세 연령 확장 대응

### 2-1. 한도 운영 기본 방향

- 기존 시니어 담보(~70세) 기준을 **기본 베이스**로 사용
- 당사 손해율 및 업계 운영기준 감안하여 **한도 조정이 필요한 담보만 별도 기준 적용**
- **업계 공시 리서치 결과**: 71~75세 구간별 담보 한도는 공개 공시에 노출되지 않음 (내부 UW 가이드라인 영역) → 공개 벤치마킹 불가, 자사 손해율 기반 자체 기준 수립 필요
- **연령 상한 업계 현황**: 공개 노출 단독상해 상품 다수가 70세 상한. 75세까지 공개 노출된 단독상해 상품 미확인 → 당사 71~75세 확장은 업계 대비 선제적 포지션
- **규제 환경**: 노후·유병력 실손 가입연령 90세 확대(2025.4.1 시행, 금융위) — 고령 인수 완화 흐름이 정책적으로 진행 중이라는 점은 본 확장의 우호적 근거

### 2-3. 71~75세 한도 조정 필요 담보 (손해율 감안 검토 우선순위)

- 고연령일수록 청구 빈도·강도 증가가 예상되는 담보를 우선 검토
- 특히 골절 관련 담보: 70대 이상에서 골절 청구 집중 경향
- **공개 벤치마킹 불가 확인** → 자사 손해율 역산 + 보험다모아 실시간 견적 샘플링(70세 입력 후 담보별 한도 노출 확인) 방식으로 대체 검토 권고

> **리서치 결론**: 타사 71~75세 담보별 한도는 공개 공시 미노출 — 비공개 1차 자료(GA 인수지침, 기초서류) 확보 경로로 전환 또는 보험다모아 견적 샘플링 병행 필요

#### 합산한도 운영 검토 포인트

- 당사 사고건 기준 주말 응급실 이용 비율 약 **44%**
- 복수 응급실 담보 동시 가입 시 이중 수령 가능 구조 → 합산 한도 설계 필요

| 구분 | 내용 |
|---|---|
| 기존 담보 (1~3등급) 단독 한도 | 1등급 기준 ~60세 100만 / 60~75세 20만 |
| 신규 담보 A 단독 한도 | 응급실내원 예상 최대 20만 |
| 신규 담보 B 단독 한도 | 평일심야/공휴일 예상 최대 10만 |
| **신규 담보 A + B 동시 가입 시 합산** | **최대 30만 노출 — 합산 통합 한도 설계 필요** |
| **기존 담보 + 신규 담보 동시 가입 시 합산** | **기존 1등급 + 신규 A+B = 최대 130만 노출 가능 — 통합 한도 설계 필요** |

## 4. 인수기준 완화 검토 요청 사항

> **주**: 이 섹션은 내부 검토 요청 사항 구조화용이며, 의사결정 근거는 업계 리서치 + 당사 손해율 확인 후 결정

### 4-1. 스코어링 기준 완화 (골절조건 제거 검토)

| 항목 | 내용 |
|---|---|
| 요청 내용 | TM 매출실적 이슈로 강화된 기준을 완화 검토 |
| 구체적 내용 | 스코어링 기준에서 골절 조건 제거 가능 여부 검토 |
| 검토 필요 사항 | 골절 조건 제거 시 예상 인수 증가 규모 및 손해율 영향 추정 |
| 업계 현황 | 스코어링 기준은 공개 공시 미노출 (전형적 내부 UW 영역) — 업계 공개 비교 불가 |
| 의견 | 71~75세 확장과 골절 조건 제거가 동시에 적용될 경우 고연령 골절 청구 집중 구조와 맞물려 손해율 악화 위험 상당. 확장과 완화를 분리 검토하거나 단계적 적용 고려 필요 |

### 4-2. 골절진단비 시니어 한도 상향 (~500만원 예상)

| 항목 | 내용 |
|---|---|
| 요청 내용 | 시니어(70세 이상) 골절진단비 한도 상향 |
| 예상 상향 폭 | 현행 → 약 500만원 수준 (기초서류 확인 필요) |
| 검토 필요 사항 | 당사 70세 이상 골절 청구 손해율, 업계 동일 연령 한도 수준 비교 |
| 업계 현황 | 골절진단비 연령별 한도는 공개 공시 미노출 — "500만원이 업계 표준"인지 공개 검증 불가. 보험다모아 실시간 견적 샘플링 또는 GA 인수지침 입수를 통한 확인 권고 |
| 의견 | 고령 골절 청구 집중 구조 감안 시 상향 수준 신중 검토 필요. 500만원 상향 시 당사 손해율 역산 근거 필수 |

### 4-3. 상해수술비 한도 상향

| 항목 | 내용 |
|---|---|
| 요청 내용 | 시니어 구간 상해수술비 한도 상향 검토 |
| 검토 필요 사항 | 당사 상해수술비 손해율 추이, 업계 동일 연령 한도 수준 비교 |
| 업계 현황 | 상해수술비 연령별 한도 공개 공시 미노출 — 업계 비교 불가 |
| 의견 | 상향에 신중한 입장 유지 — 상해수술비는 단독 청구 외 연계 청구 가능성 있고, 71~75세 신규 연령 구간 손해율 데이터가 미확보 상태에서 한도 상향은 추가 리스크 요인. 연령 확장 안정화 이후 재검토 방안 제안 |
